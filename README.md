# 🚗 Car Price Prediction System

## 📌 Overview

This project focuses on predicting the **selling price of used cars** using **Machine Learning techniques**.

The system analyzes various features such as:

* Car brand
* Year of manufacture
* Fuel type
* Kilometers driven
* Ownership history

and predicts the **estimated market price** of the car.

This type of project is widely used in real-world applications to help both **buyers and sellers make better decisions** ([UpdateGadh][1])

---

## 🚀 Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature engineering
* Car price prediction using regression models
* Model evaluation and performance analysis
* Data visualization

## 📊 Machine Learning Models Used

* Linear Regression
* Decision Tree Regressor
